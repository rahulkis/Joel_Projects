Create Engaging Conversational Bots. We help you make Chatbots that can be used on your Website. No coding required. Engage every single visitor on your site. 

== Description ==

TARS is the simplest way to make and put a chatbot on your website.
You can make complex and powerful automated conversations with TARS Bot Builder, in minutes.
Put them on your website and automatically start conversation with any visitor coming to your site.
 

== Installation ==

1. Login to your WordPress Admin  (E.g. http://yourWordpressWebsite/wp-admin/). Select Plugins. Click on Add New option.
2. Get new account from <a target="_blank" href="https://admin.hellotars.com/#register">TARS</a>. 
3. Copy the snippet code to 'Settings > TARS Bot Widget'.

Need help? <a target="_blank" href="http://help.hellotars.com/distribute/adding-convbot-widget/adding-bot-widget-on-wordpress-site">Read Install Instructions</a>or <a target="_blank" href="https://www.youtube.com/watch?v=QdlZkxfuGNA">Watch a short tutorial</a>
