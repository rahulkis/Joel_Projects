=== Add-on Gravity Forms - MailPoet 3 ===

MailPoet is a free newsletter and post notification plugin for WordPress that makes it really simple to send out email newsletters to your subscription lists. This plugin integrates Gravity Forms with MailPoet by providing an option for your customers to signup for your newsletter lists while submitting a form.

== Description ==

> #### Requirements
> [MailPoet 3](https://wordpress.org/plugins/mailpoet/)  
> [Gravity Forms](http://www.gravityforms.com)

Create MailPoet Newsletter subscribers from Gravity Forms entries.

Requires Gravity Forms version 1.9.16 or higher and MailPoet 3.0 or higher.

== Installation ==

1. Install as a regular WordPress plugin
2. Create a form with the appropriate fields via Gravity Forms
3. Setup Mail Poet Newsletters with one or many lists
4. Navigate to a form then select MailPoet under Settings and setup your feeds

== Features ==

* Allow your users to sign up for a Mailpoet newsletter list using a gravity form
* You can capture first name, last name, and (of course) email
* You can signup users to as many lists as you like
* You can set up the form to opt in or opt out
* Ability to add mailpoet custom field

= How to integrate mailpoet custom field in gravity form =

First edit any Mailpoet form and add a custom field
<http://prntscr.com/ntrzbt>

Now create/edit a gravity form and add an exact field
<http://prntscr.com/nts0cw>

Now hover on the "MailPoet custom field ID" tooltip and copy the ID. Paste that ID to the "MailPoet custom field ID" field
<http://prntscr.com/nts26o>

That's it. Now gravity form field will be synchronized with MailPoet's subscriber field.
